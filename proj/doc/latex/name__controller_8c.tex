{\footnotesize\ttfamily int() letter\+\_\+input (\begin{DoxyParamCaption}\item[{unsigned char}]{code }\end{DoxyParamCaption})}



Processes a keyboard scan code into a corresponding letter. 


\begin{DoxyParams}{Parameters}
{\em code} & The scan code to process.\\
\hline
\end{DoxyParams}
This function translates a scan code into a character corresponding to a keyboard key, and adds the character to the players name input. The space bar adds a space to the name, and backspace deletes the last character from the name.

\begin{DoxyReturn}{Returns}
Returns 0 upon successful letter conversion, and 1 if the letter is the same as the previous keystroke. 
\end{DoxyReturn}
