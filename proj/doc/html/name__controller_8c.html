◆ letter_input()

int() letter_input ( unsigned char  code)

Processes a keyboard scan code into a corresponding letter.

Parameters
codeThe scan code to process.

This function translates a scan code into a character corresponding to a keyboard key, and adds the character to the players name input. The space bar adds a space to the name, and backspace deletes the last character from the name.

Returns
Returns 0 upon successful letter conversion, and 1 if the letter is the same as the previous keystroke.

int  size

Processes keyboard inputs within the player name context.

Parameters
scan_codeAn array of scan codes to process.
sizeThe size of the scan_code array.

This function listens for keyboard inputs when in the player name state. It changes the state to MENU when the ESC key is pressed, and to GAME when the ENTER key is pressed. For any other key press, it calls letter_input function to process the keystroke.
