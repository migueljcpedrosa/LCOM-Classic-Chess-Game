Extracts the least significant byte from a word.

Parameters
valThe word to extract the LSB from.
lsbA pointer to the variable that will hold the extracted LSB.

This function extracts the least significant byte (LSB) from a word (val) and stores it in the provided pointer (lsb). It returns an error if the pointer is null.
